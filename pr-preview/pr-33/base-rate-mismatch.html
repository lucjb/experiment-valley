Understanding Base Rate Mismatches

What is a Base Rate Mismatch?

A base rate mismatch occurs when the actual conversion rate observed in your experiment differs significantly from the prior estimate used to design the experiment.

Why This Matters

Higher Than Expected

  • • Loss of statistical power - Your experiment is underpowered
  • • Invalid sample size - Calculated for wrong assumptions
  • • MRE may be inappropriate - Minimum effect size needs review

Lower Than Expected

  • • Data quality issues - Tracking or technical problems
  • • External factors - Seasonal effects or market changes
  • • Confounding variables - Uncontrolled factors affecting results

Statistical Implications

Power Analysis Breakdown:

Statistical power depends on the variance of conversion rates. For a fixed absolute MRE:

  • • Higher base rates = Higher variance = Lower power
  • • Lower base rates = Lower variance = Higher power (but potential data issues)

What To Do

1. Investigate the Cause

Check for data quality issues, external factors, or changes in user behavior.

2. Recalculate Sample Size

Use the actual base rate to determine if you need more data for adequate power.

3. Consider Extending the Experiment

If underpowered, you may need to run longer or increase traffic.
